# Bookstore Module

## Description

This is a simple Odoo module to manage books and genres.

- Add books with title, author, publisher, published date, price, stock, and genres.
- Automatically calculate book age.
- Archive and unarchive books.
- Validate stock and price to avoid negative values.

## Installation

1. Copy the module folder to your Odoo addons directory.
2. Restart Odoo server.
3. Update the apps list.
4. Install the **Bookstore** module.

## Usage

- Manage genres.
- Add books and assign genres.
- Use filters to find books by genre.
- Archive/unarchive books.
